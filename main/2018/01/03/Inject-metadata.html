Method 2: Exiftool

Linux

Install exiftool

# Debian, Ubuntu, Linux Mint, ...
sudo apt-get -y install libimage-exiftool-perl

# CentOS, RadHat, Amazon Linux, ...
sudo yum install -y perl-Image-ExifTool

Add Exif XMP tag

exiftool -ProjectionType="equirectangular" IMG_0001.jpg

macOS

Install exiftool

  • macOS Package: ExifTool-10.71.dmg
  • or via brew: brew install exiftool

Add Exif XMP tag

exiftool -ProjectionType="equirectangular" IMG_0001.jpg

Windows

Install exiftool

  • Download exiftool v10.71
  • Unzip and copy the .exe file to C:\Windows\

Add Exif XMP tag

By command (Recommended)

exiftool -ProjectionType="equirectangular" IMG_0001.jpg

By GUI Exif Editor

  • Download ExifToolGUI v5.16
  • Unzip and open ExifToolGUI.exe
  1. Select your photo
  2. Click ExifTool direct
  3. Enter -ProjectionType="equirectangular" into command box below, then hit Enter
